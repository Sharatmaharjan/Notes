## PART A: USER DEFINED FUNCTION

---

### **1. Factorial Using Recursion**

**Question:**
Write a program to compute the factorial of a given number using recursion.

**Hint:**
A recursive function must have a base condition to stop further calls.

**Sample Output:**

```
Enter a number: 5
Factorial of given number = 120
```

---

### **2. Swap Two Numbers Using Call by Value**

**Question:**
Write a program to illustrate swapping of two numbers using call by value.

**Hint:**
Changes made inside the function do not affect actual arguments.

**Sample Output:**

```
Before swapping:
a = 10   b = 20
After swapping:
a = 10   b = 20
```

---

### **3. Swap Two Numbers Using Call by Reference**

**Question:**
Write a program to swap two numbers using call by reference.

**Hint:**
Use pointers to access actual memory locations of variables.

**Sample Output:**

```
Before swapping:
a = 10   b = 20
After swapping:
a = 20   b = 10
```

---

### **4. Function With No Arguments and No Return Value**

**Question:**
Write a program to demonstrate a user-defined function that takes no arguments and returns no value.

**Hint:**
Use `void` return type and input/output inside the function.

**Sample Output:**

```
Enter two numbers: 4 6
Sum = 10
```

---

### **5. Function With Arguments and Return Value**

**Question:**
Write a program to find the sum of two numbers using a function that accepts arguments and returns a value.

**Hint:**
Use a return statement to send result back to `main()`.

**Sample Output:**

```
Enter two numbers: 8 12
Sum = 20
```

Array elements are:

## PART B: RECURSION

---

### **8. Fibonacci Series Using Recursion**

**Question:**
Write a program to generate Fibonacci series using recursion.

**Hint:**
Each term is the sum of previous two terms.

**Sample Output:**

```
Enter number of terms: 6
Fibonacci series: 0 1 1 2 3 5
```

---

### **9. Sum of Digits Using Recursion**

**Question:**
Write a program to find the sum of digits of a number using recursion.

**Hint:**
Separate last digit using modulo operator.

**Sample Output:**

```
Enter a number: 1234
Sum of digits = 10
```

---

### **10. Reverse a Number Using Recursion**

**Question:**
Write a program to reverse a number using recursion.

**Hint:**
Extract digits and rebuild number recursively.

**Sample Output:**

```
Enter a number: 4567
Reversed number = 7654
```

---

### **11. Power of a Number Using Recursion**

**Question:**
Write a program to calculate power of a number using recursion.

**Hint:**
Multiply base with recursive call until exponent becomes zero.

**Sample Output:**

```
Enter base and exponent: 2 5
Result = 32
```

---

### **12. GCD Using Recursion**

**Question:**

**Hint:**
Use Euclidean algorithm.

**Sample Output:**
