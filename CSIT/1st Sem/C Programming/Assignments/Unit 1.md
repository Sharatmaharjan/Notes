### **13. Write a program to find area of a triangle when three sides are given.**

**Hint:** Use **Heron’s Formula**

* s = (a + b + c) / 2
* area = √[s(s–a)(s–b)(s–c)]
  **Sample Output:**

```
Enter sides: 3 4 5
Area = 6
```
