## **A. One-Dimensional Array**

### **1. Read and Display 10 Float Numbers**

**Question:**
Write a program to read 10 float numbers from the user and display the entered numbers on the screen.
**Hint:**
* Use a one-dimensional float array
* Use a loop for input and output

**Sample Output:**

```
Enter 10 numbers:
1.2 2.5 3.4 4.1 5.0 6.8 7.9 8.3 9.6 10.0

Entered numbers are:
1.2 2.5 3.4 4.1 5.0 6.8 7.9 8.3 9.6 10.0
```
